

# Student Result Calculator

## Objective
A responsive web application that calculates student results based on marks obtained in 5 subjects.

## Features
- Input marks for 5 subjects
- Validation (0–100)
- Total Marks Calculation
- Percentage Calculation
- Grade Calculation
- Pass/Fail Status
- Responsive Design
- Reset Functionality

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download project files.
2. Open index.html in any browser.
